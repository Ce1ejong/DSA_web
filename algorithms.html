10 Core Algorithms & Pseudocode

Each card contains the formal algorithm steps, language-agnostic pseudocode, and a working Python implementation.

1. Linear Search
Description

A sequential search algorithm that checks every element in a list until a match is found or the entire list is exhausted. Simple but inefficient for large datasets.

Step-by-step Algorithm
  1. Start from the first element of the list (index $i=0$).
  2. Compare the current element $A[i]$ with the target value $T$.
  3. If $A[i] = T$, return the index $i$ and terminate the search.
  4. Increment $i$ to check the next element.
  5. If the end of the list is reached ($i > length(A)$) and no match is found, return $-1$.
Pseudocode
FUNCTION LinearSearch(Array, Target):
    FOR i FROM 0 TO length(Array) - 1:
        IF Array[i] EQUALS Target:
            RETURN i
    RETURN -1
Python Sample Code
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

# Example:
data = [10, 5, 8, 12, 3]
print(f"Index of 8: {linear_search(data, 8)}") 
# Output: Index of 8: 2
2. Binary Search (Requires Sorted Array)
Description

An efficient search algorithm for **sorted** arrays. It repeatedly divides the search interval in half, eliminating half of the elements in each step (Divide and Conquer). [Image of Binary Search Diagram]

Step-by-step Algorithm
  1. Initialize $low$ to $0$ and $high$ to $length(A) - 1$.
  2. While $low \le high$: calculate $mid = \lfloor (low + high) / 2 \rfloor$.
  3. If $A[mid]$ equals $T$, return $mid$.
  4. If $A[mid]$ is less than $T$, set $low = mid + 1$ (search the right half).
  5. Else ($A[mid]$ is greater than $T$), set $high = mid - 1$ (search the left half).
  6. If the loop terminates, return $-1$.
Pseudocode
FUNCTION BinarySearch(Array, Target):
    low = 0
    high = length(Array) - 1
    WHILE low <= high:
        mid = (low + high) // 2
        IF Array[mid] EQUALS Target:
            RETURN mid
        ELSE IF Array[mid] < Target:
            low = mid + 1
        ELSE:
            high = mid - 1
    RETURN -1
Python Sample Code
def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

# Example (must be sorted):
data = [3, 5, 8, 10, 12]
print(f"Index of 10: {binary_search(data, 10)}") 
# Output: Index of 10: 3
3. Bubble Sort
Description

A simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. The pass-through process is repeated until no swaps are needed.

Step-by-step Algorithm
  1. Set a flag, $swapped$, to true.
  2. Repeat while $swapped$ is true: set $swapped$ to false.
  3. Iterate through the array from the first element up to $n-i-1$.
  4. If $A[j] > A[j+1]$, swap $A[j]$ and $A[j+1]$, and set $swapped$ to true.
  5. The largest unsorted element "bubbles up" to its correct position in each pass.
Pseudocode
PROCEDURE BubbleSort(A):
    n = length(A)
    FOR i FROM 0 TO n-2:
        FOR j FROM 0 TO n-i-2:
            IF A[j] > A[j+1]:
                Swap A[j] and A[j+1]
Python Sample Code
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]

# Example:
data = [64, 34, 25, 12, 22]
bubble_sort(data)
print(f"Sorted array: {data}")
# Output: Sorted array: [12, 22, 25, 34, 64]
4. Selection Sort
Description

A sorting algorithm that divides the input list into two parts: a sorted sublist and the remaining unsorted sublist. It repeatedly selects the smallest element from the unsorted sublist and moves it to the sorted sublist.

Step-by-step Algorithm
  1. Iterate from $i = 0$ to $n-2$ (where $n$ is length).
  2. Assume the element at index $i$ is the minimum (min\_idx = $i$).
  3. Iterate from $j = i+1$ to $n-1$ to find the true minimum element's index in the unsorted portion.
  4. If $A[j] < A[min\_idx]$, update $min\_idx = j$.
  5. Swap the minimum element found at $min\_idx$ with the element at index $i$.
Pseudocode
PROCEDURE SelectionSort(A):
    n = length(A)
    FOR i FROM 0 TO n-2:
        min_idx = i
        FOR j FROM i+1 TO n-1:
            IF A[j] < A[min_idx]:
                min_idx = j
        Swap A[i] and A[min_idx]
Python Sample Code
def selection_sort(arr):
    n = len(arr)
    for i in range(n - 1):
        min_idx = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i] # Swap

# Example:
data = [64, 25, 12, 22, 11]
selection_sort(data)
print(f"Sorted array: {data}")
# Output: Sorted array: [11, 12, 22, 25, 64]
5. Insertion Sort
Description

Builds the final sorted array one item at a time. It iterates, taking elements from the input data and inserting them into the correct position in the already sorted part of the array.

Step-by-step Algorithm
  1. Iterate from the second element $i = 1$ to $n-1$.
  2. Store the current element $A[i]$ as $key$.
  3. Initialize $j = i - 1$ (the last element of the sorted part).
  4. While $j \ge 0$ AND $A[j] > key$: shift $A[j]$ one position right ($A[j+1] = A[j]$), and decrement $j$.
  5. Insert $key$ into the correct position: $A[j+1] = key$.
Pseudocode
PROCEDURE InsertionSort(A):
    n = length(A)
    FOR i FROM 1 TO n-1:
        key = A[i]
        j = i - 1
        WHILE j >= 0 AND A[j] > key:
            A[j+1] = A[j]
            j = j - 1
        A[j+1] = key
Python Sample Code
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and key < arr[j]:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key

# Example:
data = [12, 11, 13, 5, 6]
insertion_sort(data)
print(f"Sorted array: {data}")
# Output: Sorted array: [5, 6, 11, 12, 13]
6. Factorial (Iterative)
Description

Calculates the factorial of a non-negative integer $n$ (denoted $n!$). It is the product of all positive integers less than or equal to $n$. Uses an iterative (looping) approach.

Step-by-step Algorithm
  1. If $n < 0$, return error. If $n = 0$, return 1.
  2. Initialize $result = 1$.
  3. Iterate from $i = 1$ to $n$.
  4. In each iteration, update $result = result \times i$.
  5. Return $result$.
Pseudocode
FUNCTION IterativeFactorial(n):
    IF n < 0: RETURN Error
    IF n = 0: RETURN 1
    result = 1
    FOR i FROM 1 TO n:
        result = result * i
    RETURN result
Python Sample Code
def iterative_factorial(n):
    if n < 0: return "Error"
    if n == 0: return 1
    result = 1
    for i in range(1, n + 1):
        result *= i
    return result

# Example:
print(f"Factorial of 5: {iterative_factorial(5)}")
# Output: Factorial of 5: 120 (5*4*3*2*1)
7. Fibonacci Sequence (Recursive)
Description

A sequence where each number is the sum of the two preceding ones, usually starting with 0 and 1. This uses a recursive approach, which is simple but computationally expensive.

Step-by-step Algorithm
  1. Define the base cases: If $n = 0$, return $0$. If $n = 1$, return $1$.
  2. For $n > 1$, the function recursively calls itself: $Fib(n) = Fib(n-1) + Fib(n-2)$.
  3. The calculation continues until the base cases are reached.
Pseudocode
FUNCTION RecursiveFib(n):
    IF n = 0:
        RETURN 0
    ELSE IF n = 1:
        RETURN 1
    ELSE:
        RETURN RecursiveFib(n-1) + RecursiveFib(n-2)
Python Sample Code
def recursive_fib(n):
    if n <= 0: return 0
    if n == 1: return 1
    return recursive_fib(n - 1) + recursive_fib(n - 2)

# Example:
print(f"Fibonacci of 6: {recursive_fib(6)}") 
# Output: Fibonacci of 6: 8 (0, 1, 1, 2, 3, 5, 8)
8. Greatest Common Divisor (GCD)
Description

Calculates the largest positive integer that divides two or more integers without leaving a remainder, using the efficient Euclidean Algorithm.

Step-by-step Algorithm
  1. Let $a$ and $b$ be the two numbers.
  2. While $b \neq 0$:
  3. Calculate the remainder $r = a \pmod b$.
  4. Set $a = b$.
  5. Set $b = r$.
  6. When $b$ becomes $0$, $a$ is the GCD. Return $a$.
Pseudocode
FUNCTION GCD(a, b):
    WHILE b IS NOT 0:
        remainder = a MOD b
        a = b
        b = remainder
    RETURN a
Python Sample Code
def gcd(a, b):
    while b:
        a, b = b, a % b
    return a

# Example:
print(f"GCD of 54 and 24: {gcd(54, 24)}")
# Output: GCD of 54 and 24: 6
9. Prime Number Check (Optimization)
Description

Determines if a given positive integer $n$ is a prime number. A prime number is greater than 1 and has no positive divisors other than 1 and itself. The optimization limits checking divisors up to $\sqrt{n}$.

Step-by-step Algorithm
  1. If $n \le 1$, return False.
  2. If $n \le 3$, return True.
  3. If $n$ is divisible by 2 or 3, return False.
  4. Iterate from $i = 5$ to $\lfloor \sqrt{n} \rfloor$ (incrementing by 6).
  5. If $n$ is divisible by $i$ or $i+2$, return False.
  6. If the loop completes, return True.
Pseudocode
FUNCTION isPrime(n):
    IF n <= 1: RETURN False
    IF n <= 3: RETURN True
    IF n MOD 2 = 0 OR n MOD 3 = 0: RETURN False
    i = 5
    WHILE i * i <= n:
        IF n MOD i = 0 OR n MOD (i + 2) = 0:
            RETURN False
        i = i + 6
    RETURN True
Python Sample Code
import math
def is_prime(n):
    if n <= 1: return False
    if n <= 3: return True
    if n % 2 == 0 or n % 3 == 0: return False
    
    i = 5
    while i * i <= n:
        if n % i == 0 or n % (i + 2) == 0:
            return False
        i += 6
    return True

# Example:
print(f"Is 29 prime? {is_prime(29)}") 
# Output: Is 29 prime? True
10. Sum of Array Elements
Description

Calculates the sum of all elements within an array or list using a simple iterative loop. This is a fundamental traversal algorithm.

Step-by-step Algorithm
  1. Initialize a $sum$ variable to $0$.
  2. Iterate through each $element$ in the Array.
  3. Update $sum = sum + element$.
  4. After iterating through all elements, return $sum$.
Pseudocode
FUNCTION SumArray(Array):
    sum = 0
    FOR element IN Array:
        sum = sum + element
    RETURN sum
Python Sample Code
def sum_array(arr):
    total = 0
    for element in arr:
        total += element
    return total

# Example:
data = [1, 2, 3, 4, 5]
print(f"Sum of array: {sum_array(data)}")
# Output: Sum of array: 15
