Explanation

In Python, the built-in list type acts as a dynamic array. The core operations are implemented using basic list methods (like .insert() and .pop()) wrapped in a class for structure. Traversal simply involves iterating through all elements.

Python Code (`python_scripts/array.py`)
class ArrayOperations:
    def __init__(self, initial_list=None):
        self.array = initial_list if initial_list is not None else []
        # Output: Initial Array: [10, 20, 30, 40]

    def insert_element(self, index, value):
        self.array.insert(index, value)

    def delete_element(self, index):
        self.array.pop(index)

    def traverse(self):
        return self.array

# Example Usage:
arr_ops = ArrayOperations([10, 20, 30, 40])
arr_ops.insert_element(2, 50)  # Insert 50 at index 2
arr_ops.delete_element(3)     # Delete element at index 3 (was 30)
current_array = arr_ops.traverse()
Output Example
Initial Array: [10, 20, 30, 40]
Array after inserting 50 at index 2: [10, 20, 50, 30, 40]
Array after deleting element at index 3: [10, 20, 50, 40]

Explanation

A Queue is a FIFO (First-In, First-Out) structure, like a line of people waiting. **Enqueue** adds an element to the rear, and **Dequeue** removes an element from the front. We use Python's `collections.deque` for an efficient implementation.

Python Code (`python_scripts/queue.py`)
from collections import deque

class Queue:
    def __init__(self):
        self.items = deque()

    def enqueue(self, item):
        self.items.append(item) # Add to the rear

    def dequeue(self):
        if self.items:
            return self.items.popleft() # Remove from the front
        return "Queue is empty"
    
    def display(self):
        return list(self.items)

# Example Usage:
q = Queue()
q.enqueue(10)
q.enqueue(20)
q.dequeue() 
current_queue = q.display()
Output Example
Initial Queue: []
After Enqueue (10, 20, 30): deque([10, 20, 30])
After Dequeue (removes 10): 20
Current Queue: [20, 30]
